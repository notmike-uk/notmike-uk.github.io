Module mysql

MySQL server.

require("module/mysql")

local myDatabase = mysql:Database{database='mydatabase'}
myDatabase:Grant{user='god', password='thisisasecurepasswordxkcd'}

An auto-generated root password is stored in the mysql directory as .mysql-root.pwd.
